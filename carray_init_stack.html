NAME

carray_init_stack - initialise a new array on the stack

SYNOPSIS

#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <cware/carray/carray.h>

#define carray_init_stack(array, buffer, _capacity, namespace);

DESCRIPTION

The carray_init_stack macro initialises a new array on the stack. The resulting array does not need to be released from memory at the end of the program, as it is on the stack, and will be automatically freed as its caller goes out of scope. However, because of this, the array cannot grow in size. It will be stuck at the fixed number of elements given to it.

This macro will print an error message to the stderr, and exit the program when any of the following conditions are met:

•

The buffer is NULL

•

_capacity is less than or equal to zero

array will be the array variable to initialise the array into
buffer will be the buffer to use to store the contents of the array in
_capacity will be the maximum capacity of the list
namespace will be the namespace of the array settings to use

EXAMPLES

/*
* This example showcases how to use the carray_init_stack macro.
*/

#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <cware/carray/carray.h>

struct StringArray {
int length;
int capacity;
char **contents;
};

int main(void) {
/* Stack arrays must be passed as a pointer to the functions that operate
on them */
char *strings[4] = {0};
struct StringArray array = carray_init_stack(&array, strings, 4
STRING_ARRAY);

/* No need to free the array; do note the use of a & though. */

return EXIT_SUCCESS;
}
