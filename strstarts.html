NAME

strstarts - determine if a string is prefixed with a sub-string.

SYNOPSIS

#include <cware/libstr/libstr.h>

int strstarts(const char *string, const char *start);

DESCRIPTION

The strstarts function will determine if a string is prefixed with a sub-string. Or, in other words, if a string starts with another string.

If string or start are NULL, then the program exits with an error message printed to the stderr.

string will be the string to check the start of
start will be the sub-string to search for

RETURN VALUE

This function returns 1 if the string is prefixed with the sub-string, and 0 if the string is not prefixed with the substring.

EXAMPLES

/*
* This example showcases how to use the strstarts function.
*/

#include <stdio.h>
#include <stdlib.h>
#include <cware/libstr/libstr,h>

int main(void) {
printf("’foo’ starts with ’f’: %i\n", strstarts("foo", "f"));
printf("’foo’ starts with ’fo’: %i\n", strstarts("foo", "fo"));
printf("’bar’ starts with ’fo’: %i\n", strstarts("foo", "fo"));

return EXIT_SUCCESS;
}

CONFORMING TO

This function conforms to C89.
