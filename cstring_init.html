NAME

cstring_init - initialize a new cstring

SYNOPSIS

#include "cstring.h"

struct CString cstring_init(const char *base);

DESCRIPTION

Initialize a new cstring with a base string in the body of the cstring. If an empty cstring is desired, then base should be an empty string.

This function will display an error to the stderr, and abort the program if any of following conditions are met.

o base is NULL

base will be the base string to use as the body

RETURN VALUE

This function will return a new cstring

NOTES

This function has no notes.

EXAMPLES

#include "cstring.h"

int main(void) {
struct CString new_string = cstring_init("basic body");

cstring_free(new_string);

return 0;
}
