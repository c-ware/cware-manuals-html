NAME

libstr - an extension to string.h

SYNOPSIS

#include <stddef.h>
#include <cware/libstr/libstr.h>

char *strrev(char *string);
int strstrip(char *string, const char *characters);
int strcount(const char *string, const char *count);
int strstarts(const char *string, const char *start);
int strrepl(char *string, const char *find, const char *replace, size_t size);

DESCRIPTION

The libstr library provides an extension to the string(3) library. It provides additional functions which operate on NUL terminated strings that the base string library does not provide. What follows is a table of each function that is exposed by the public headers, and where documentation can be found for it.

CONFORMING TO

This library conforms to C89.
