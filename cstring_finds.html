NAME

cstring_finds - find a c-style string inside of a cstring

SYNOPSIS

#include "cstring.h"

int cstring_finds(struct CString haystack, const char *needle);

DESCRIPTION

Determine the location of a needle in a haystack, where the needle is a C-style string as opposed to a CString, and the haystack is a CString.

This function will display an error to the stderr, and abort the program if any of following conditions are met.

o haystack.contents is NULL
o haystack.length is negative
o needle is NULL

haystack will be the cstring to search through
needle will be the c-style string to find in the haystack

RETURN VALUE

This function will return the location of first character of needle, or CSTRING_NOT_FOUND

NOTES

This function has no notes.

EXAMPLES

#include <stdio.h>

#include "cstring.h"

int main(void) {
struct CString string_a = cstring_init("foo bar baz");

printf("Location of ’bar’: %i0, cstring_finds(string_a, "bar"));

cstring_free(string_a);

return 0;
}
