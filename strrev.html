NAME

strrev - reverses a string in-place

SYNOPSIS

#include <cware/libstr/libstr.h>

char *strrev(char *string);

DESCRIPTION

The strrev function will reverse a string in-place. This means that the function does not allocate any extra space during the reversing, and instead opts for shifting parts of the buffer around.

When string is NULL, an error will be printed to the stderr, and the program will exit.

string will be the string that will be reversed.

RETURN VALUE

This function will return the string that was reversed.

EXAMPLES

/*
* This example showcases how to use the strrev function.
*/

#include <stdio.h>
#include <stdlib.h>
#include <cware/libstr/libstr.h>

int main(void) {
char string[256 + 1] = "foo bar baz";

strrev(string); /* Reverse the string */
printf("%s\n", string);
strrev(string); /* Put it back to normal */

return EXIT_SUCCESS;
}

CONFORMING TO

This function conforms to C89.
