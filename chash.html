NAME

SYNOPSIS

int length; /* How much of the hashtable has been filled */

DESCRIPTION

An ANSI C compliant hashtable implementation using macros. Allows for type safety, and tries to be as memory efficient as possible. The hashtable implementation allows for the hashtable to be used as a stack or heap hashtable which can allow for quick lookups of values without having to use the heap.
