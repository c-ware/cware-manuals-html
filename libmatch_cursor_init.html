NAME

libmatch_cursor_init - initialize a new list

SYNOPSIS

#include "libmatch.h"

struct LibmatchCursor libmatch_cursor_init(char *buffer, int length);

DESCRIPTION

The libmatch_cursor_init function will initialize a new cursor using a pre-allocated buffer. The buffer can either be on the stack, or the heap. If a heap buffer is used, then the cursor will either have to be freed manually, or be released with the libmatch_cursor_free(cware) function.
