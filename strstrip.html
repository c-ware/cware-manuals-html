NAME

strstrip - strip characters from a string in-place

SYNOPSIS

#include <cware/libstr/libstr.h>

int strstrip(char *string, const char *characters);

DESCRIPTION

The strstrip function will remove characters within its characters argument from string in-place. This means that the function does not allocate any extra space during the removal of characters, and instead opts for shifting parts of the buffer around. A misconception about this function may be that it treats characters as a string to remove, when in reality those are just the characters that will be removed from the string. The final string will be NULL terminated.

If the string or characters arguments are NULL, an error will be printed to the stderr, and the program will exit.

string will be the string to remove characters from
characters will be the characters to remove

RETURN VALUE

This function returns the number of characters that were stripped.

EXAMPLES

/*
* This example showcases how to use the strstrip function.
*/

#include <stdio.h>
#include <string.h>
#include <stdlib.h>
#include <cware/libstr/libstr.h>

int main(void) {
char string[256 + 1] = "foo bar baz";

printf("Stripped %i characters from string.\n", strstrip(string, "fb"));
printf("%s\n", string);

return EXIT_SUCCESS;
}

CONFORMING TO

This function conforms to C89.
