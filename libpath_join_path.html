NAME

libpath_join_path - join a variable number of path segments into a single buffer

SYNOPSIS

#include "libpath.h"

int libpath_join_path(char *buffer, int length);

DESCRIPTION

Join a variable number of path components into a single string. All path components will automatically have path segments put between them which match the corresponding path for the system it is running on. The final path parameter MUST be NULL, and the final string should be large enough to have a NUL byte written to it.

This function will display an error to the stderr, and abort the program if any of following conditions are met.

o buffer is NULL
o length is negative

buffer will be the buffer to write to
length will be the length of the buffer

RETURN VALUE

This function will return the number of bytes written

NOTES

This function has no notes.

EXAMPLES

#include "libpath.h"

int main(void) {
char buffer[1024 + 1];

libpath_join_path(buffer, 1024, LIBPATH_HOME_DIR, "config", NULL);

return 0;
}
