NAME

cstring - ansi c compliant dynamically allocated string

SYNOPSIS

/* a dynamically allocated string */
struct CString {

int length; /* the length of the string */
int capacity; /* the physical capacity of the string */
char *contents; /* the characters in the string */

};
/* an array of cstrings-- integration with carray(cware) */
struct CStrings {

int length; /* the length of the array */
int capacity; /* the physical capacity of the array */
struct CString *contents; /* the strings in the array */

};

struct CString cstring_init(const char *base);
void cstring_free(struct CString cstring);
struct CString cstring_loadf(FILE *file);
void cstring_reset(struct CString cstring);
int cstring_finds(struct CString haystack, const char *needle);
int cstring_find(struct CString haystack, struct CString needle);
int cstring_strip(struct CString *cstring, struct CString target);
int cstring_strips(struct CString *cstring, struct CString target);
int cstring_endswiths(struct CString cstring, const char *check);
int cstring_endswith(struct CString cstring, struct CString check);
void cstring_concats(struct CString *cstring, const char *string);
int cstring_startswiths(struct CString cstring, const char *check);
int cstring_startswith(struct CString cstring, struct CString check);
void cstring_concat(struct CString *cstring_a, struct CString cstring_b);
struct CString cstring_slice(struct CString cstring, int start, int stop);

DESCRIPTION

An ANSI C compliant dynamically allocated string implementation with a focus on efficient use of memory. One of the main design choices of cstring is to put emphesis on the reuse of existing string objects to reduce overall memory usage.

Another important design choice of cstring is to provide some degree of interopability between a cstring, and a C-style string. To achieve this, cstrings are NUL terminated, while also having a length field for any situations where the programmer may want length-prefixed strings, such as constant time length operations, and the need for the NUL byte in the body of the string.
