NAME

cstring_startswiths - determine if a cstring starts with a c-style string

SYNOPSIS

#include "cstring.h"

int cstring_startswiths(struct CString cstring, const char *check);

DESCRIPTION

Determine if a CString starts with a C-style string. In other words, is a C-style string the prefix of another CString.

This function will display an error to the stderr, and abort the program if any of following conditions are met.

o cstring.contents is NULL
o check is NULL
o cstring.length is negative

cstring will be the cstring to check
check will be the c-style string to check for

RETURN VALUE

This function will return 1 if cstring starts with check, 0 if it does not

NOTES

This function has no notes.

EXAMPLES

#include <stdio.h>

#include "cstring.h"

int main(void) {
struct CString string_a = cstring_init("foo bar baz");

if(cstring_startswiths(string_a, "foo") == 1)
printf("%s", "string_a starts with ’foo’0);
else
printf("%s", "string_a does not start with ’foo’0);

cstring_free(string_a);

return 0;
}
