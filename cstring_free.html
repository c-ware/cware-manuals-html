NAME

cstring_free - release a cstring from memory

SYNOPSIS

#include "cstring.h"

void cstring_free(struct CString cstring);

DESCRIPTION

Releases a cstring from memory. Since the cstring structure is most likely going to be on the stack, only the body of the string must be released.

This function will display an error to the stderr, and abort the program if any of following conditions are met.

o cstring.contents is NULL

cstring will be the cstring to free

RETURN VALUE

This function has no return value.

NOTES

This function has no notes.

EXAMPLES

#include "cstring.h"

int main(void) {
struct CString new_string = cstring_init("basic body");

cstring_free(new_string);

return 0;
}
