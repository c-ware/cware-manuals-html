NAME

cstring_loadf - load a file into a cstring

SYNOPSIS

#include "cstring.h"

struct CString cstring_loadf(FILE *file);

DESCRIPTION

Loads the contents of a file into a NUL-terminated cstring. Do note, that this function will NOT work on streams. So, this will not work on stdin, for example.

This function will display an error to the stderr, and abort the program if any of following conditions are met.

o file is NULL

file will be the file to load

RETURN VALUE

This function will return a new cstring

NOTES

This function has no notes.

EXAMPLES

#include "cstring.h"

int main(void) {
// Load a file into a string
struct CString string_a = cstring_loadf("./file.txt");

cstring_free(string_a);

return 0;
}
