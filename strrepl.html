NAME

strrepl - in-place string find and replace

SYNOPSIS

#include <cware/libstr/libstr.h>

int strrepl(char *string, const char *find, const char *replace, size_t size);

DESCRIPTION

The strrepl function replaces all occurrences of find in string with replace, in-place. This means that the function does not allocate any extra space during the replacement, and instead opts for shifting parts of the buffer around. This function is also intended to work with stack arrays. Because of that, this function will not, and cannot expand the buffer to make room for large replacements. Instead, strrepl will .I not perform the replacement if the final size is beyond the capacity of the buffer.

When string, find, or replace are NULL, an error will printed to the stderr, and the program will exit. An error will also occur if the string argument has a longer length than the size of the buffer, determined by size.

string will be the string buffer to replace in
find will be the string to replace
replace will be what find is substituted with
size will be the maximum length of the buffer

RETURN VALUE

This function returns the number of times find was replaced. If the replacement could not be performed, which would be because of the buffer’s size being too small, then -1 will be returned.

EXAMPLES

/*
* This example showcases how to use the strrepl function.
*/

#include <stdio.h>
#include <string.h>
#include <stdlib.h>
#include <cware/libstr/libstr.h>

int main(void) {
char string[256 + 1] = {0};

strcat(string, "foo bar baz");
strrepl(string, " ", "__", 256);

printf("%s\n", string);

return EXIT_SUCCESS;
}

CONFORMING TO

This function conforms to C89.
