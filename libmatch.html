NAME

libmatch - collection of string parsing utilities

SYNOPSIS

DESCRIPTION

The libmatch(cware) library provides a collection of generic string parsing utilities. The intention is to make parsing strings, and reporting errors in the formatting of these strings easier. The functions provided by the library are centered around a cursor object, which is used as a container for information like the line and character number, the position in the string, and more. This allows for incremental parsing of a string.

The library is quite useful for projects such as

•

Programming or configuration language parsing

•

Finding patterns in strings (similar to regex)

As well as much more. The libmatch(cware) library is organized into groups of functions. You can find the functions that are available to you in the following sections.
